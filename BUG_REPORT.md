# 🐛 الإبلاغ عن المشاكل

## كيفية الإبلاغ عن خطأ أو مشكلة

شكراً لمساعدتك في تحسين التطبيق! إذا واجهت مشكلة، يرجى:

### 1. افتح Issue جديد
انقر على [Issues](https://github.com/gamemosa89-spec/App-ai-beta-/issues) ثم "New Issue"

### 2. وصف المشكلة بوضوح
- **العنوان**: اكتب عنوان واضح للمشكلة
- **الوصف**: اشرح ماذا حدث بالتفصيل
- **الخطوات**: اكتب خطوات لإعادة إنتاج المشكلة
- **السلوك المتوقع**: ماذا كان يجب أن يحدث
- **السلوك الفعلي**: ماذا حدث بدلاً منه
- **لقطات الشاشة**: أضف صور توضح المشكلة إن أمكن

### 3. معلومات إضافية
- نظام التشغيل (Windows, Mac, Linux)
- متصفح الويب والإصدار
- أي رسائل خطأ

### مثال على Issue جيدة:
```
العنوان: خطأ عند تحميل الصور

الوصف:
عندما أحاول تحميل صورة بصيغة PNG، يظهر رسالة خطأ

الخطوات:
1. أدخل API Key
2. انقر على "إنشاء صورة"
3. اكتب وصف
4. انقر على "تحميل"

السلوك المتوقع:
يجب أن يتم تحميل الصورة بنجاح

السلوك الفعلي:
يظهر رسالة "خطأ في التحميل"

متصفح: Chrome 125
النظام: Windows 11
```

---

شكراً على الإبلاغ! 🙏
